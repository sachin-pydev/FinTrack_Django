**Fintrack** is a Django-based web application designed to help users manage personal finances efficiently. It tracks income, expenses, and budgets, providing visual insights to make smarter financial decisions.

---

## Key Features

### Dashboard
- **Quick financial summary:**  
  - Average daily expense  
  - Total income  
  - Total expense  
  - Total balance
- **Visual analytics:**  
  - Income vs Expense (Pie Chart)  
  - Expenses by Category (Bar Chart)  
- Displays the last five added transactions for quick reference

### Records Page
- View complete transaction history with summary cards  
- Filter records by selected time period  
- Add, edit, or delete transactions  
- Download PDF of transactions in selected date range

### Budget Page
- Create, edit, and manage multiple budgets  
- Delete or add new budgets  
- View transactions linked to specific budgets
